*****  UI Shopping Cart automation pipeline using Python + Selenium and Jenkins  *****
It was created based on fake shopping cart for testing from https://www.saucedemo.com/
The code was developed using VSCode tool.
You can clone and open the folder on VSCode and click play button or open terminal type "python main.py" and press Enter.
The pipeline runs once a day.
You can see the last run automation report on https://luckxander.github.io/shop/
